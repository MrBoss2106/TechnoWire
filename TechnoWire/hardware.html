CPU / Processor

Ook wel bekend als CPU, hij is instaat voor basisbewerkingen en -controle bij het uitvoeren van programmacode.

Moederbord

Een moederbord is een printplaat met elektronica waarop andere (insteek-)printplaten kunnen worden gemonteerd.

Geheugen / RAM

Het geheugen is een onderdeel van een computer waarin tijdelijk informatie opgeslagen kan worden voor later gebruik.

Opslag Media

Dit type of opslag is geheugen dat de opgeslagen informatie kan bewaren, ongeacht of de computer is in- of uitgeschakeld.

Videokaart / GPU

Dit is een grafische processor, die gebruikt wordt voor alle videotaken. Hiermee neemt hij deze taken van de CPU over.

Voeding

De voeding zorgt ervoor dat de onderdelen van een computer niet teveel stroom krijgen.
